# Scraping & API: Pornhub & Onlyfans

Этот проект собирает ссылки на социальные сети (Twitter, Instagram, OnlyFans, TikTok), метрики и текстовые признаки профилей с Pornhub и Onlyfans, обрабатывает данные и создаёт структурированный набор данных для анализа. Итоговый датасет сохраняется в формате CSV, готовый для EDA.

## Описание датасета

> **Примечание**: Ссылки на датасет и подробное описание будут добавлены здесь.

Датасет (`[ВПИСАТЬ НАЗВАНИЕ]`) содержит ~[ВПИСАТЬ ЧИСЛО] строк, каждая из которых представляет отдельную модель. Столбцы:

- `name`: Имя профиля
- `twitter`: URL Twitter/X
- `instagram`: URL Instagram
- `onlyfans`: URL OnlyFans
- `tiktok`: URL TikTok
- `twitter_exists`: есть ли Twitter/X
- `instagram_exists`: есть ли Instagram
- `onlyfans_exists`: есть ли OnlyFans
- `tiktok_exists`: есть ли TikTok
- `videos_count`: Количество загруженных видео
- `views_count`: Общее количество просмотров профиля
- `average_views`: Среднее количество просмотров
- `num_platforms`: Общее колчество платформ
- `tags`: список топ-3 популярных категорий
